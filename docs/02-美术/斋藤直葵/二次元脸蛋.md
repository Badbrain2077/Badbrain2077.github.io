# 二次元脸蛋

> [【日语中字】如何将脸画得更可爱？什么是「印象值」？（Saitou Naoki）](https://www.bilibili.com/video/BV1GB4y1M7Yz/?spm_id_from=333.788.recommend_more_video.6)

具体参照**真实人类发育**的五官特征总结

---

## 脸颊

1. 越平,下巴越尖,人物越成熟,冷静年长
2. 越圆,下巴越圆,人物越可爱,幼稚年幼
3. 腮红:
     - 渐变:浅红,深色感觉疲惫生病,黑眼圈
     - 斜线:长短,疏密
     - 作用:可爱,害羞,兴奋

## 眼睛

([详见](二次元眼睛))

1. 越高,越成熟;越低,越孩子气
2. 越远,越神秘,不可思议;越近越有吸引力
3. 越大,印象越深刻;越小,越真实
4. 睫毛:
     - 越细,越纤细;越厚,越吸引目光
     - 中间:元气活泼
     - 两侧:成熟色气
     - 下睫毛:色气(真实化妆)
5. 阴影
      - 多,抑郁;少,活泼
6. 高光
      - 偏观众,在上方:吸引
      - 远离观众,偏下方:弱化
      - 总之类型很多

## 鼻子

(美少女弱化,或独特风格)

1. 越高,越成熟;越低,越孩子气
2. 高鼻梁:成熟
3. 鼻孔:真实,丑陋
4. 画法:
      - 鼻顶面线
      - 鼻底面线
      - 高光
      - 阴影
      - 不画
5. 离嘴越远,越猴子

## 嘴巴

1. 具体情绪口型
2. 口鼻间距越大,越猴子
3. 画在中线内侧,如果在中心像是突起的撅嘴

## 头发

优先级很高,直接铺体积,具体见[头发](二次元头发)

1. 头发厚薄,决定头发浓密和稀疏
2. 构成,一级型,二级型,三级型:体块,发梢,分布
3. 细节适量,走结构
4. 侧发靠近,显脸小

## 思想

1. 大盒子:构图,整体规划,把上面的位置都定好了,**不要抠细节!**
2. 小盒子:根据具体模块的特征进行类型选择
3. 细化

## Tip

1. 灰底好减弱观察误差诱导
2. 快照过程图
3. 复制保留侧边过程图
