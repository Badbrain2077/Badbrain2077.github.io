微软日语输入法，使用方法。

- 用 x 或者 l 加上罗马字来输入小写的假名，例如 xa 表示小写的ぁ，li 表示小写的ぃ。
- 用 F6 键来转换平假名，用 F7 键来转换片假名，用 F8 键来转换半角片假名，用 F9 键来转换全角英文数字，用 F10 键来转换半角英文数字。
- 用 Alt + CapsLock 键来切换片假名输入，用 Ctrl + CapsLock 键来切换平假名输入 12。
- 长音符号“-”
- 促音 ：双重输入后一发音的第一个字母后，会出现促音，比如：切符，きっぷ kippu、学校 がっこう gakkou  。（单独输入小っ的时候可以使用直接输入 ltu 或者 xtu）
