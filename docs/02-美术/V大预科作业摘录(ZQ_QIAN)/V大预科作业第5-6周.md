## 〓【色】

### ★【色彩感觉】

![img](https://i0.hdslb.com/bfs/article/80996f52308bb808abd5909100ba578b65af0326.png@1320w_960h.webp)

- 很多人对色彩有误区、总觉得自己没有色感、实际上可以通过一些练习来改善你的颜色控制能力

- 之前说了色彩三要素、色相、明度、纯度、不管是什么图、都只是色相、明度、纯度组成的

- 不要把画画看的太玄幻、它只是一种对于图像的理解方式、色彩感觉也是一样的

- 除非你是医学上的色弱和色盲、这确实是没有办法的、但是只要你眼睛是正常的、那都是可以通过练习来改善色彩感觉的

- 首先把明度排除在外、因为明度是素描的部分、所以如果你控制不了色彩、无非就是对色相和纯度控制不了

- 定义一张图色彩感觉好、或者色彩感觉不好的标准其实是没有定死的、但是可以理解为标准就是对色相和纯度的控制能力、实际上是对比的控制能力

- 当我们说感觉的时候、就说明大部分人建立了一个共识、所以当所有人都觉得不好的时候我觉得好、那我觉得好就没有意义、因为我们做设计的要获得观众的认可才有意义、这也是设计和纯艺的小差别

![img](https://static.hdslb.com/images/transparent.gif)

- 左边是火星、右边是地球、大部分人都会觉得地球的色彩感觉好、因为现在人类的眼睛已经完全适应了地球的环境、天空是蓝的、阳光是偏暖的等等、这种概念就是人类的认知

- 火星的颜色不说丑、但是感觉怪怪的、就是因为人类没有在火星生存过、无法适应这种天空和地面、所以也没有冷暖的概念

### ★【冷暖对比】

![img](https://static.hdslb.com/images/transparent.gif)

- 只要画面里存在好的对比、冷暖对比和纯度对比、一般来说画面的颜色就不会很差

![img](https://static.hdslb.com/images/transparent.gif)

- 同样是黄色系、但也还是有一个更暖或者更冷的概念、所以冷暖是相对关系、色环中180度的两个颜色是极冷和极暖、但没有意义

- 橙色、不改变色相只降低饱和度之后就变冷了、蓝色、不改变色相只降低饱和度之后就变暖了、所以冷暖不止是由色相决定的、也和饱和度有关系

![img](https://static.hdslb.com/images/transparent.gif)

- 两个颜色在红黄区间、但是从感觉上来说有冷暖概念

- 冷暖对比色相有一部分关系、但是饱和度更加重要

- 理解了这个概念之后、就不会想画暖色就去选暖色的色相、想画冷色就去选冷色的色相、一般来说色感好的人饱和度也控制的好

![img](https://static.hdslb.com/images/transparent.gif)

- 这张图里没有冷色的色相、但是低饱和的颜色一样起到了冷色的作用、因为和旁边的高饱和暖色起到了对比

- 临摹的时候也要注意这个概念、这个看起来像暖色或者冷色的颜色、到底是什么样的颜色

### ★【色彩倾向】

![img](https://static.hdslb.com/images/transparent.gif)

- 如果你的画面没有色彩倾向说明你的画面没有组织起来、这种颜色很难让人留下深刻的映像、同样的色彩倾向也是可以练习的

- 原图其实就是阴天白光的环境、色彩也很丰富、当我调色之后、虽然整张图颜色都和原图不一样了、但是色彩感觉还在、还是有冷暖对比、怎么调整都很好看

- 当你原先的冷暖已经做好了、这种情况调色没问题、反过来说如果图像本身色彩就有问题的、就很难调整好看的颜色

- 所以练习就是找一张好看的原图、进行主观的颜色倾向、要求是倾向完之后还是要保留色彩的冷暖对比、要做到整体颜色都倾向一个颜色的时候、冷暖对比还在

- 练习色彩倾向之后、在创作的时候也可以用上

![img](https://static.hdslb.com/images/transparent.gif)

- A和B其实是一样的、但是B有了色彩倾向

![img](https://static.hdslb.com/images/transparent.gif)

- 左图阳光打下来、固有色有了黄色的色彩倾向、右图整体倾向蓝青色、但是在这种整体倾向的同时、我们还是能辨认物体本身的颜色、比如说红色的帽子、白色的衣服

- 所以当你学会了颜色倾向之后、对色彩的控制力会上升、对色彩的推理也会得心应手

## 〓【色彩倾向作业】

![img](https://static.hdslb.com/images/transparent.gif)

- 做色彩倾向练习很多时候会调动感受、不要搞得过于理性

- 接下来说的方法不一定是完全正确的、只是一个思路参考、而且色彩倾向本身就没有正确答案

- 原图有低饱和颜色、冷色和暖色、倾向色是红色

- 原图低饱和的颜色、倾向的就会更加明显

- 原图与倾向色饱和度接近、色相为同类色的、稍微向倾向色移动、提高饱和度

- 原图与倾向色饱和度接近、色相为对比色的、小幅度向倾向色移动或者不移动、可能会降低饱和度

- 原图明度高的颜色、变化会很少

- 这个练习没有什么标准答案、只要保证冷暖对比就行

![img](https://static.hdslb.com/images/transparent.gif)

- 首先用之前讲的思路、大概推算一下原图倾向之后每个颜色会变成什么样、左边往蓝青色走、右边往紫红色走

- 有一些过于关注素描关系的同学在色彩方面会有点障碍、不是因为色感不好、是因为太在意标准了、色彩不要太纠结、多调动感受

- 目前这个底色相对于原图来说一个偏冷、一个偏暖、但还是有对比在里面

- 不要一直去调整明度、这是关于色相和饱和度的练习、想办法通过调整色相和饱和度来倾向颜色、当然明度不要和原图差太多

![img](https://static.hdslb.com/images/transparent.gif)

- 因为原图色彩感觉就不错、我参考原图做色彩倾向也一样不错

- 原图里的暖色、在偏暖的倾向提高了饱和度、在偏冷的倾向降低了饱和度、但是在偏冷的整体来看降低了饱和度它也还是暖色

- 建议这个练习不要画的太慢、太慢了大脑会进入理性状态

- 尽量调动感受来画出不同的色调、但是要表现出原图的基本关系

## 〓【透视基础+结构基础2】

### ★【结构翻转思路】

- 虽然是简单的结构翻转、但是勉勉强强做得到是不行的、要做到非常熟练才能在创作中用的起来、所以需要量的积累

- 画一张创作、要把某个参考放进去、如果你放不进去说明你透视和结构不过关

![img](https://static.hdslb.com/images/transparent.gif)

- 之前说过所有东西都要放进盒子里面、如果不放进盒子里什么点都定不出来

![img](https://static.hdslb.com/images/transparent.gif)

- 但是有很多同学连最开始的盒子都画不好、盒子的比例和参考不一样、所以在画盒子之前可以画一个草图获得一个感性认知

![img](https://static.hdslb.com/images/transparent.gif)

- 先画一个盒子、把物体大概的画一个草图放进盒子看看比例、如果比例不对就可以调整盒子

![img](https://static.hdslb.com/images/transparent.gif)

- 主体紧贴包裹、之前说过了盒子要刚好包裹住物体、主体紧贴、也就是盒子包裹住最主要的主体、一些结构或者细节再慢慢加上去

- 分解为若干单元、每个加上去的结构都可以看成是独立的、用盒子去包裹每个单元、同时忽略细节用几何体去理解、然后定位这些单元在主体的哪个位置、再点定位画出必要的结构

- 如果能做到这里差不多就准了、新建图层勾线

## 〓【结构翻转作业】

- 因为是结构翻转的作业所以角度要发生变化、不要用一样的角度

- 注意分层、每一层处理一部分这样比较方便、最后再勾线

![img](https://static.hdslb.com/images/transparent.gif)

- 首先分析物体、画几张立面图、拆分物体分析每个单元、有一些小结构和细节可以直接忽略归到一个单元、有一些弧度之类的也不需要管、等主体画好了之后再加上去

- 确定盒子的角度、画一个草图、定下视平线和消失点、画出盒子

![img](https://static.hdslb.com/images/transparent.gif)

- 盒子下面的角不要小于90度、防止畸变、然后勾一遍线

- 定位主体盒子和单元盒子的交界点、注意透视

![img](https://static.hdslb.com/images/transparent.gif)

- 用之前讲的技巧、计算每个单元盒子和主体盒子的关系、可以多用估算

- 这一步其实是最难的一步、这一步没画好后面很多东西都是乱的

- 如果没办法理解计算这个空间盒子、可以先画一个草图试一试

![img](https://static.hdslb.com/images/transparent.gif)

- 新建图层勾线

- 结构翻转越到细节的地方越可以靠感觉、越大的地方越不能出错、比如透视比例严重失调

![img](https://static.hdslb.com/images/transparent.gif)

- 有必要的地方就用点定位、没有必要的地方就靠估算和手绘、比如一些圆角、一些小结构

- 注意斜面或者弧面上面的结构是跟着斜面弧面的透视走的

![img](https://static.hdslb.com/images/transparent.gif)

- 现在草稿基本上已经画完了、可以开始勾线了、没有把握的同学可以先画一个稍微草的线稿或者标记一些记号再勾线

- 注意打了圆角的线比包裹的盒子线要往里收一些

- 这个练习在前面的部分更加重要、勾线没必要这么细、大概的比例正确、有细节就可以了

## 〓【抽象对比分析入门】

- 很多人不会看图、都是凭直觉看图、当然自觉没有错、但是你要做专业人士的话还是要学会看图的技巧、并且做出分析

![img](https://i0.hdslb.com/bfs/article/fb0209d9db9a07ce2c4a8b2c3643850352d9d703.png@1320w_702h.webp)

- 不要把它看成是具象的物体、只把它当作图形或者颜色、一定要忘记物体本身是什么、它只是形状和颜色

- 抽象观察能看出好的作品为什么强调某个部分、或者是让某个部分变得更好看、也就是分析别人是用了什么方法让作品成立的、也就是外行看热闹、内行看门道

- 要看清楚别人的招、你才能学到东西、否则你只是觉得这张图好看、画的好像、光影很好等等、但是为什么好看你说不出来

- 比如说一个破旧的墙、上面长了一些发霉的东西之类的、但是你可能会觉得它颜色不错、比干净的白墙还好看、就是因为上面的图形和颜色组合出来的抽象图形或者概念让你感觉好看、所以学抽象就是为了用这个工具去审美

### ★【抽象对比】

![img](https://i0.hdslb.com/bfs/article/3adbe1fef7993f5ea2119727446b711912f8a7f4.png@1320w_504h.webp)

- 平均分布状态、在这个区域里面的元素是缺乏对比的

- 左上角的空间、左下角的人头、右上角的灯光都是平均分布状态、因为没有对比

![img](https://i0.hdslb.com/bfs/article/79ec30418f4a5e9693351e516ae377b3102963d4.png@1320w_740h.webp)

- 线性分布状态和曲线分布状态都有点像渐变、只不过曲线渐变的会更夸张一点

- 线性分布状态就像排身高、平均分布就是每个人都一样高、线性分布每个人身高相差的不多、最高的这个人还是不容易产生关注

![img](https://i0.hdslb.com/bfs/article/5d9cd2c3ce81b09ebf25040926aa9e493d207dde.png@1320w_508h.webp)

- 曲线分布状态就是一群人里有一个特别高、其它的突然矮下去、它就可以产生关注

![img](https://i0.hdslb.com/bfs/article/ee63f994a8f3aee32c24b34bffbc04df98f94565.png@1320w_440h.webp)

- 带噪声的非线性分布状态、这张图里有一个最明显的重点、她被白光照到、饱和度偏低、其他的东西都在暖色里面、但是排除掉她之后、剩下的东西还有对比、这就是噪声、这是我自己提出的概念

- A本身有对比、当A和B在一个空间的时候、A就成为了一个整体、A和B是最强的对比、A本身的对比也变成了弱对比

![img](https://static.hdslb.com/images/transparent.gif)

- 这四种对比状态可以分析所有画面

- 至于这些柱状可以是任何东西、比如大小、颜色等等

- 一般来说我们希望让画面整体的效果是第四种状态、但是前面三种状态也会在画面中出现、但是整体效果还是第四种、有一个重点的同时、其他部分也有节奏

### ★【抽象分析步骤】

![img](https://static.hdslb.com/images/transparent.gif)

- 我们画图就是要有重点、所以不要找没重点的图分析

![img](https://static.hdslb.com/images/transparent.gif)

- 首先看画面里有没有一个吸引你的元素、在哪里、是什么、一般来说这两张图就是人物这块区域会吸引人

- 因为我们能够观察到这个特别的重点、所以它和其它地方肯定是有区别的、所以找到它和其它地方有区别的原因

- 右图人物暗、背景亮、所以是明度的对比、左图人物清晰、背景模糊、所以是边缘的对比

![img](https://static.hdslb.com/images/transparent.gif)

- 除了画面重点区域的对比、找到其它次要的对比

- 左图近景人物形状大和其它人物形状小的对比、或者暖色的遮阳布和冷色的天空的对比、这些对比都可以、找到这些理由

- 右图左下角对比最弱、往上走弱、往上走稍强、往右走更强的对比关系、但是这些对比都要比人物这个区域的对比要更弱

- 从这前三点来看得出结论、第一要求有重点、第二要求重点之外的节奏、大部分好的叙事性的画、一眼看完整张图的画都是这个概念

![img](https://static.hdslb.com/images/transparent.gif)

- 最后分析画面的节奏都是什么样的抽象对比形式、好看的图都是高手打架、所以可以把经验积累起来

- 右图的云从多变少是线性分布、天空的冷往下变暖是线性分布或者曲线分布、取决于冷暖变化的速度有多快

- 左图人物的裤子是平均分布

- 有一些同学画了很长时间的图了还是会犯的一个毛病、就是把画面里每一个对比都画的太强

- 比如画这个裤子的时候会把里面的什么东西都画出来、比如褶皱之类的、但是你会发现你画出来还不如不画出来会更好看、有的时候就让它是一个平面的、平均分布状态反而会更好看

- 因为这个裤子是平均的时候、它会和画面中的其它东西产生对比、类似的还有右边的船、如果你把船的明度提亮把细节都画出来、这个照片的氛围一下子就没有了、它就是要有这种对比关系画面才会好看

![img](https://static.hdslb.com/images/transparent.gif)

- 这张图第一眼就看过去的重点是红衣女子、当然每个人看到的重点可能不一样、但是基本上都在这一块

- 重点对比红衣女子高饱和度、周围低饱和度

- 次要对比右边柱子那一块和左边这一块、左边这一块区域差不多是一个平均分布、我们的视线没法在这一块地方上停留、这实际上是一个高手的做法、你不管怎么看这块地方也看不到什么东西、因为作者就不想让你看这块地方、人的注意力是有限的、作者希望你去看他想让你看的东西、也就是所谓的引导观众的视线

- 这张图放大之后每个小地方都会有这些对比、当然功力不够的人是做不到的

![img](https://static.hdslb.com/images/transparent.gif)

- 这张图第一眼的重点是白马、因为周围的马包括后面的人都是偏红的、所以可以说是明度对比、或者饱和度对比、或者色相的对比

- 地面的草如果画的过于对比强烈、栩栩如生、这张图就不好看了

- 所以有了强对比之后、就要有弱对比、也就是地上的草

- 我现在说的这些分析、完全是我对这张图的解读、这些解读进行的越深入、它就有可能会变成我自己去画图的时候的工具、也就是吸取这张图的经验

![img](https://static.hdslb.com/images/transparent.gif)

- 这张图的重点就过于明显了、它有色相对比、大小对比

- 一张图如果你找不到平均分布和线性分布、那那张图肯定不好看、只强不弱、那种图是不会好看的、当然纯艺术除外、和我们没关系
