# 透视基础

绘画的知识没有那么复制,基本几句话就可以讲清楚的.

很多教学视频讲不清楚,因为他们水平不行.

绘画是手艺向技能,能画出来最重要: **画不出来就是不会画,少JB废话.jpg**

花时间的大多是练习练习再练习,所以有一些实践笔记是可以的.

> 以精简准确为准.

## L1-透视导论

本质上,用**贴贴乐,连连看**实现三维空间物体结构在二维平面上的位置展示,实现大脑空间感调动.

表现为:结构,明暗分隔,投影,重叠穿插(前中后景),消失点形式

常用技巧:

- X交线法-求中点
- N字倍增法
- 透视网格贴图
- 软件透视尺
- 3D辅助([方块](https://sketchfab.com/3d-models/--c4b168bff6de4327968daa2283846044))
