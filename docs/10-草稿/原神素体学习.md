# 原神素体学习

学习知识点：

- 躯干 2
- 下肢 3 / 3-4（高挑）



头身比

| 成年女性 / 成年男性     | 7头 上下 |
| ----------------------- | -------- |
| 少女 / 少年             | 6头 上下 |
| 萝莉 / 正太（暂无） | 5头 上下 |
| 幼儿（派蒙）            | 4头      |

简而言之，头身比越大，角色越成熟，越小越可爱。

胖子、老人另算。

动物、怪物另算。



## 合理确认身形的步骤

1. 确定头身比
2. 头部大小确定
3. 头以下 2  ：3+分割
4. 上半身1  ：1
5. 下半身1.5  ：1.5+

`小腿长一点显得修长`
