# 画面结构

> [【日语中字】优秀的画有这3个特征，配图讲解！（Saitou Naoki）](https://www.bilibili.com/video/BV1Fy4y1i7oJ)

让画面抓眼球,显得专业,就要运用人视觉的性质.

1. 空间感
   1. 物体符合基本透视
   2. 有前景,中景,背景
   3. 近大远小,近实远虚
   4. 夸张透视
   5. 穿插
   6. 孔隙
2. 有重点(构成:色彩构成&几何构成)
   1. 趋势线引导向中心
   2. 黑白对比最强在中心
   3. 饱和度
   4. 虚实对比
   5. 形态对比
   6. 疏密对比
   7. 构图法
3. 有想法
   1. **心态要放正,相信自己**
   2. 不自暴自弃
   3. 认真对待
   4. 多多修改
