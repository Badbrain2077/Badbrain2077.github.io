# 关于网络对话现象的想法

## 分工细化的这些年

自从人类迈入农业社会以来 ，依赖技术的不断进步，知识储量不断积累，社会生产力的增长速度越来越来越快，个人掌握一个领域的大部分知识的时间不断在延长，为了适应社会劳动的需要，个人不得不投入更多的时间在更少更深的领域中，以习得用于工作的专业技能。

原本我们的祖先在成年的时候就可以掌握有关生存的大部分技能，生火，采集，储存，狩猎，逃跑，避开各种危险，他们已然成为这个广大领域的专家，而现在我们的极度细分领域的博士一般也要30岁出头。

在此基础上产生的工种细分，让工人的可替代性不断上升，抗风险能力被削弱，从而越来越离不开社会的生存资源的供给，并且是偏向掌控者的，所以剥削也是越来越厉害······

至此便不再深入了······在一定的时间内,由于相近领域的信息主动或被动涌入个人的知识体系，其他知识的储量就会减少，而正确全面的信息（知识）塑造了我们的认知，在理想中信息水平的的进步是和平的，但是不同认知的人们的相互敌视，还是让人十分失望。
