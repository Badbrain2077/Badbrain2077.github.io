# PS先进生产力

很多自动化的功能一定要用,拥抱  **先进生产力** ,比如:

- 动作
- 批处理
- 形状(库)
- 图案(网点,材质填充)

> 记得导出保存,备份!
>
> 命名好,可搜索!

## 动作

1. 窗口-动作
2. 录制
3. 命名-导出保存

> 目前好用的动作:
>
> - 白底背景
> - 2倍画布（右）
> - 黑色描边图层
> - 黑白线稿抽出
> - 渐变映射
> - 临摹

## 形状

1. 画好图案
2. 选取工具-右键-建立工作路径(矢量)
3. 编辑-定义自定形状
4. 命名-导出保存

> 常用的人物,建筑,山脉的纯色剪影…
>
> 准星,logo,透视网格,飘带背景
>
> 栅格化使用(或许)

## 图案

1. 画好图案
2. 编辑-定义图案
3. 命名-导出保存

> 无限空间的填充,靠选取来约束.
>
> - 网点
> - 材质
> - 更精细一些的重复图案背景(现做)
