# 健身知识

- 减脂只取决于最终摄入能量和消耗能量差值。
- 合适的重量，4组12次

- 多吃蛋白质
- 多吃蔬菜 / 不要多吃水果
- 多喝水
- 锻炼完后30分钟内喝蛋白粉效果最好
-  可以力量训练之后去做轻中度有氧训练
- RM： 如何选择重量和组数，次数
  - ＜6 RM：绝对力量
  - 6-12 RM：长肌肉
  - ＞12 RM：耐力
  - 3 - 4组
  - 一个部位一周练2-3次，要休息
