# 绘画实力和段位水平【抖抖村】

> 来来来，让我们复习一下╮(￣▽￣)╭

- 初段：基本靠兴趣，没有正规基础训练
建议：先培养兴趣，不要被吓到，要记住这段时期喜欢画画的热情

- 二段：线条开始流畅，人体结构还是不行，手脚画不好，怎么都临摹不像，上但是色更干净
建议：开始接触构图、透视等基础素描训练

- 三段：开始有整体构图的意识，掌握了点装饰画面的小技巧，人体开始有一定理解但画出来还是有些不自然，上色喜欢涂抹，临摹不错，但自己画就崩，不会找参考，没有细节，常识积累也不够
建议：需要真的去补足人体结构等基础

- 四段：整体画面更干净整洁，构图更完整，笔触更漂亮，开始注重细节刻画，光影大体无错，但姿态和重心还是经常出问题，经常说自己画风不稳定的一般也出现在这个阶段
（从事3D动画行业最好拥有三、四段的画功）
建议：大量补足基础训练，或找好的老师给予反馈和指导，自己的审美也要大幅提高

- 五段：独当一面，专业级领域，养家糊口
开始被分配到动画原画、卡通设计、主美、导演等团队管理的岗位，可能此生就这样了
建议：强化自己的风格

- 六段：能在圈子里积累较多粉丝受众，独立支撑一整本插画集，只要做好推广，工作会自动来找你
建议：继续强化风格，并加强其他领域的常识积累，画风和名字都是好招牌，取个好名字。（可能真正的dalao都从不会在意名字吧😄……），需要专注于更有效的表达

- 七段：大师级画手，空窗期不多

- 八段：史诗级画手
建议：提高自己的审美，做自己的导师
