# how to draw

记住，照着定好的目标来做，不要一时兴起，画一些无关紧要的设计来浪费自己宝贵的时间。

设定目标可以用最古老最有效的方法：作笔记。把点子写在纸上，然后随身带着，只要一有想法就把它记下来，临时要写生时，这些笔记就可以拿来提醒自己。如果是工作需求时，笔记也可以让自己容易忽略的地方，人难免会犯一些相同的错误，找一个方法提醒自己会对未来的职涯有相当大的帮助。下面我们就来示范一次，如何从设定目标到完成图稿。

先做一个如上述的目标设定可以帮助你在设计上少走一些弯路。因为在笔记过程中你的大脑就已经开始在思考了，等目标设定完，你可能就经有不少点子了，我称这种为“正向发想”，当你脑中有点子时，你可以试着在脑中开始一步一步的作画，把大致的结构都在脑中跑一次，这样你的创意会更加明确与合理，等实际画在纸上时，你可以更专注的去解决那些小细节与变化。

哪里要放置引擎、人员要坐在哪里、行李要如何收纳、轮距要多宽、车辆要多高或是车体要离地多高等等。
