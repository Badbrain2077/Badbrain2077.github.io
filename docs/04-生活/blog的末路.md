# blog的末路

- 流量的转移和聚集:微信公众号,知乎,CSDN,微博,头条...
- 内容多元化:视频化,图文结合:公众号,BILIBILI,简书..
- 给人运营经济成本高,时间成本高,收益低
- 代替品强大:功能完善,简洁,反馈迅速,节约精力

相对的

- 用别人的,迁移成本高,数据更不安全
- 自定义不足,只能专注于核心内容(其实也挺好)

如何走

- 博客写作练手,要是以后成本过高可以关闭
- 可以多平台发布,即在知乎,公众号,微博,bilibili专栏,简书,CSDN发布,又在自己的博客上发布
- 并做好多云盘的文件备份,好迁移.(虽然我并不知道相应的法规,能不能多平台发布,能不能关闭盈利共这么干,应该是有的吧)
