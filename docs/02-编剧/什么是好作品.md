# 什么是好作品?

什么是好作品?什么是好故事?

漫画,动画,电影,电视剧,PV,小说,游戏,短视频……

读者不是傻子,没有人喜欢金玉其外,败絮其中的故事.

## 摘抄

**没把观众当小孩。**

**没把观众当傻子。**

**每一个角色都在倾尽全力做好自己该做的事。**

**每一个角色都遵循着他们自己的性格和设定推动着故事的前进。**

**该发生的事和他们每一个人的抉择都清晰地在一个统一完整的世界背景观下合理地交互。**

**即使是对英雄联盟完全不了解的人看后也会被剧情引导着让心情随之起伏跌宕。**

**这就是一部优秀的动画该有的样子。**
