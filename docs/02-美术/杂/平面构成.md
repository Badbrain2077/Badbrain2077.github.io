# 平面构成

[平面构成](https://baike.baidu.com/item/%E5%B9%B3%E9%9D%A2%E6%9E%84%E6%88%90/1272?fr=aladdin)

> 定义:
>
> 平面构成是视觉元素在二次元的平面上，按照美的视觉效果，力学的原理，进行编排和组合，它是以理性和逻辑推理来创造形象、研究形象与形象之间的排列的方法。是[理性](https://baike.baidu.com/item/理性/576586)与[感性](https://baike.baidu.com/item/感性/3453796)相结合的产物。
>
> 研究在二维平面内创造理想形态，或是将既有的形态（具象或抽象形态）按照一定原理进行分解，组合，从而构成多种理想的视觉形式的造型设计基础课程。
