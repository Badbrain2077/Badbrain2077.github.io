# Excel科研

## excel科研作图

**原则：**

- 先把数据整理好，分组，命名
- 理解数据，明白这个图重点想展示什么信息
- 寻找高效自动化的方法

**技巧：**

- 运用表格模板，主流的4种，柱状图，折线图，散点图，三线表格
- 录制运用宏
- 规范格式标准
- 符合场景需求

**格式：**

- 简约，无杂线
- 去处自带的x，y坐标网格
- x,y坐标左下对齐，或者完整黑色边框
- x,y轴内部标尺
- x,y轴黑色，线粗1px
- 图例悬浮
- 颜色，强烈对比，高饱和，互补色，形状不一样，或者就黑色
- 字体黑色，中文宋体，英文新罗马，字号10.5

> 不同要求，具体查百度

## excel数据处理

**原则：**

- 先理解好数据的意义
- 规划好数据的用法
- 流程图

**技巧：**

- 用好默认
- 公式ctrl+D
- 录制宏+快捷键
- 根据类型标颜色
- 对齐好看
- 图表对齐Alt
- 分类编组，良好命名，汇总，迭代编号，做好备份
- 理解数据模型

**其他：**

- 添加到快捷栏
- 用插件：方方格子...
- 简单表格用协同办公：石墨文档，WPS，腾讯文档...
